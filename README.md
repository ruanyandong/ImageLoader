# ImageLoader
#### ImageLoader实现图片墙，具备以下功能：
* 图片的同步加载（从内存中直接加载）
* 图片的异步加载（用线程池从磁盘或者网络加载）
* 图片压缩（使用图片采样率进行压缩）
* 内存缓存（LruCache）
* 磁盘缓存（DiskLruCache）
* 网络加载（BitmapFactory.decodeStream）
* 给ImageView设置url的tag，防止列表加载乱序
#### 优化列表的卡顿现象
* 核心：不要在主线程中做太耗时的操作即可提高列表的滑动流畅度
* getView方法中不要做耗时操作
* 控制异步任务的执行频率，在列表滑动的时候不要加载，停止时加载
* 开启硬件加速，充分利用GPU特性，使界面渲染更加平滑，但会消耗更多内存。
